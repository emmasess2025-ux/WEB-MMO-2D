# WEB-MMO-2D
WEB-MMO-2D is a fully-featured, browser-based MMORPG currently in active development. Built with a custom HTML5 Canvas rendering engine and a highly optimized authoritative Node.js server via WebSockets.

The project includes complex, full-blown game systems such as real-time PvP combat, persistent inventory and currency, user authentication, and social mechanics (guilds and friend lists). The core engine is complete, and the project is currently in the map-building phase, gearing up for its first playable beta release.
